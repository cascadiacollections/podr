A podcast player for the web.

## Local development

1. Git clone the repository `gh repo clone cascadiacollections/podr`.
1. Ensure yarn is installed `npm i -g yarn`
1. Run `yarn install`.
1. Run `yarn start` to launch the local web server with live reload.
